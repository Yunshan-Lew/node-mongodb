# node-MongoDB

通过nodeJS框架Express操作MongoDB数据库，包含增（add.js）、删（delete.js）、查（get.js）、改（change.js）、等功能，供初次接触MongDB的朋友参考学习。

关于MongoDB的下载与更多信息请参考：https://www.mongodb.com/

## 运行

### 安装依赖包

```sh
npm run install
```

### 启动增加服务

```sh
node add
```

### 启动删除服务

```sh
node delete
```

### 启动查询服务

```sh
node get
```

### 启动修改服务

```sh
node change
```
